An Introduction to this class

In this course, we will attempt to create webistes that will be super presentable to our peers. In addition, this is a basic language that nearly all beginners learn, so it is absolutely necessary that we learn HTML first. Plus, it will teach basic programming skills and habits for later on!

Our Schedule

This will be what we will be doing for the next 3 months!

  • October: Website Design with HTML
  • November: Game Design with Java
  • December: Video Editting with KdenLive

Our Rules

Here are a set of rules so everyone has fun coding!

  1. When the teacher is talking, no talking
  2. If you have a question, ask the teacher
  3. If you are done with the activity, tell the teacher
  4. Help each other! Don't make fun of each of another for not getting something; we all learn at different paces!
  5. Try to come to every session
  6. Think before you code
  7. Have fun!
